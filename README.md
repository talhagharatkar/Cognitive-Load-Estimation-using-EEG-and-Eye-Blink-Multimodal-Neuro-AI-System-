# Cognitive-Load-Estimation-using-EEG-and-Eye-Blink-Multimodal-Neuro-AI-System-
In real-world scenarios like driving, learning, air-traffic control, and healthcare, excessive mental workload leads This project measures how hard the brain is working by watching eye blinking and brain signals, and then uses AI to decide whether the person is relaxed, focused, or mentally overloaded.
